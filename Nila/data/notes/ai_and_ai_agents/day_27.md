## Day 27: AI and AI Agents Debugging and Review
### Key concepts
- The core idea of AI and AI Agents debugging and review is to identify and fix errors in the code. A simple exercise in this context would be to use a debugger to step through the code and identify the source of the error. Common beginner mistakes include not using a debugger and not testing the code thoroughly.
### Examples
```python
import pdb

def add_numbers(a, b):
    pdb.set_trace()
    return a + b
```
### Practice notes
- Use a debugger to step through the code and identify the source of the error.
- Test the code with different inputs and refine it.
