## Day 23: AI and AI Agents Core Building Blocks
### Key concepts
- The core idea of AI and AI Agents core building blocks is to understand the fundamental components of AI agents. This involves understanding the simple exercise of creating an agent, identifying common beginner mistakes such as not defining the agent's goals, and learning from them to improve the project.
### Examples
```python
# Example of a core building block of an AI agent
import random

class Agent:
    def __init__(self):
        self.goals = ['move', 'jump', 'shoot']
    def act(self):
        return random.choice(self.goals)
```
