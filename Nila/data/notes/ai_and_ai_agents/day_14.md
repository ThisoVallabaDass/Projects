## Day 14: AI and AI Agents Basic Workflow
### Key concepts
- The core idea of AI and AI Agents is to create intelligent systems that can perform tasks autonomously. The basic workflow of AI and AI Agents includes problem definition, data collection, model training, and deployment. Common beginner mistakes include not defining the problem clearly and not collecting enough data.
### Examples
```python
# Example of a simple AI agent workflow
import random

class Agent:
    def __init__(self):
        self.problem = 'task_definition'
        self.data = 'data_collection'
        self.model = 'model_training'
        self.deployment = 'deployment'
    def act(self):
        return self.deployment(self.model(self.data(self.problem)))
```
