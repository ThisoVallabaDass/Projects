## Day 13: AI and AI Agents Core Building Blocks
### Key concepts
- The core idea of AI and AI Agents is to create intelligent systems that can perform tasks autonomously. The core building blocks of AI and AI Agents include perception, reasoning, and action. Common beginner mistakes include not understanding the relationships between these building blocks.
### Examples
```python
# Example of a simple AI agent with perception, reasoning, and action
import random

class Agent:
    def __init__(self):
        self.perception = 'sensor_data'
        self.reasoning = 'decision_making'
        self.action = 'move'
    def act(self):
        return self.reasoning(self.perception)
```
