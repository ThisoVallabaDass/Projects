## Day 1: What is AI and AI Agents? - The Big Picture
### Key concepts
- AI stands for Artificial Intelligence, which refers to the development of computer systems that can perform tasks that typically require human intelligence, such as visual perception, speech recognition, and decision-making. AI Agents are programs that use AI to make decisions and take actions autonomously. The problem that AI and AI Agents solve is to automate complex tasks, improve efficiency, and enhance decision-making. Real-world examples include virtual assistants like Siri and Alexa, self-driving cars, and personalized product recommendations.
### Examples
```python
# Example of a simple AI Agent
import random

class AI_Agent:
    def __init__(self, name):
        self.name = name
    def make_decision(self):
        return random.choice(['yes', 'no'])

agent = AI_Agent('Nila')
print(agent.make_decision())
```
